## What's inside

Single-file static HTML — no build step, no dependencies, no framework. Ships directly to GitHub Pages.

The design system uses CSS custom properties for theming, a monospaced + serif type pairing (Geist Mono / Instrument Serif), and a subtle grid-line background. Scroll reveals are handled with IntersectionObserver; active nav state is computed on scroll.
